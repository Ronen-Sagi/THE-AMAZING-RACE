# The Amazing Race 🏎️  

ב-The Amazing Race אתה משחק נהג מירוצים שמנסה לנצח בשלבים שונים מול בוטים ברמות קושי משתנות.  
המטרה שלך היא להשלים את המסלול מהר ככל האפשר, לזכות בכסף ולשדרג את מכונית המירוץ כדי להתמודד בשלבים קשים יותר.  
המשחק משלב כיף, תחרות ואסטרטגיה של שיפור מכונית והשתפרות בנהיגה.
